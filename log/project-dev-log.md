> 项目路径：`d:\18785\C++\`

## 第一阶段：项目初始化与基础功能

### 需求确认
- 管理员模块：供应商管理、商品管理、员工管理、会员管理、采购管理（主表+明细）
- 普通用户模块：个人信息、商品信息、采购信息
- 数据库：MySQL + 建表 SQL 脚本，前端 UI：Element Plus
- 会员管理选择基础 CRUD 方案

### 后端搭建
- Spring Boot 2.7.18 + JDK 17，Maven 构建
- 6 张数据表：supplier、employee、product、member、purchase_main、purchase_detail
- 6 个 JPA Entity + 6 个 Repository，含模糊搜索方法
- 6 个 Service + 7 个 Controller（含 AuthController）
- JWT 认证：登录生成 token（24h），AuthInterceptor 校验，admin 接口 level=1 权限控制
- BCrypt 密码加密，统一响应 Result {code, msg, data}
- 跨域 CORS 配置，全局异常处理
- 批量导入：支持竖线 `|` 分隔，逐行解析

### 前端搭建
- Vue 3.4 + Vite 5 + Element Plus 2.5 + Vue Router 4 + Pinia + Axios
- Axios 拦截器：自动附 token，401 跳登录，错误提示
- 路由守卫：无 token → /login，非 admin → /user，admin → /admin
- 14 个页面组件（登录、管理员 6 页、用户 4 页、布局 2 页）

### 初始数据
- 管理员 admin/123456，普通用户 user001/123456，员工 EMP001/123456
- 3 家供应商、5 件商品、3 名会员、3 笔采购单 + 4 条明细
